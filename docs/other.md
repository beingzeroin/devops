## MkDocs

Documentations

* [MkDocs](http://www.mkdocs.org)

Install
```bash
pip install mkdocs
```

Setup in current directory
```bash
mkdocs new .
```

Start dev server with hot reload on [http://127.0.0.1:8000](http://127.0.0.1:8000)
```bash
mkdocs serve
```

Build static site
```bash
mkdocs build --clean
```

Deploy to github
```bash
mkdocs gh-deploy
```
